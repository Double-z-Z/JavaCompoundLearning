# 学习推荐

> **触发时机**: 
> - 用户说"开始今日学习"
> - 每次新建对话时（作为开场分析）
> - 完成一个主题后
>
> **数据来源**:
> - `.agent/profile.md` - 学习者画像、已掌握知识点、mastery分数
> - `02-Knowledge/` - 知识库扫描
> - `03-Practice/mistakes/` - 错误模式
> - `04-Maps/MOC-*.md` - 主题地图
> - `01-Projects/` - 进行中的项目

---

## 工作流程

### 1. 数据分析

读取并分析以下数据：

```
.agent/profile.md
├── 已掌握知识点及mastery分数
├── 当前进行中的项目
├── 历史学习数据（学习频率、偏好）
└── 当前学习计划（用户上次的决定）

02-Knowledge/<主题>/concepts/
├── 每个概念的mastery分数
├── 最后更新时间
└── 入链/出链数量（网络密度）

03-Practice/mistakes/
├── 活跃错误模式（未完全掌握）
└── 错误发生频率

04-Maps/MOC-*.md
├── 各主题学习进度
└── 核心概念掌握状态

01-Projects/
└── 进行中的项目及当前Phase
```

### 2. 生成学习状态分析

输出格式：

```markdown
📊 **当前学习状态分析**

### 进行中的项目
- [[项目名]]（Phase X）
  - 涉及知识点: [[概念A]]、[[概念B]]
  - 当前Phase目标: ...
  - 查看 [[MOC-主题]] 了解整体进度

### 最近学习（7天内）
| 概念 | mastery | 学习时间 | 状态 |
|-----|---------|---------|------|
| [[概念A]] | 65 | 2天前 | 🌿理解 |
| [[概念B]] | 45 | 5天前 | 🌿理解 |

### 待复习（艾宾浩斯曲线）
| 概念 | mastery | 上次学习 | 推荐动作 |
|-----|---------|---------|---------|
| [[概念C]] | 55 | 7天前 | 完成1个练习 |
| [[概念D]] | 40 | 10天前 | 回顾+练习 |

### 知识网络健康度
- 知识孤岛: [[概念E]]（缺乏关联）
- 高频枢纽: [[概念F]]（被多次引用）
- 网络密度: X%（目标: 80%）
```

### 3. 生成学习推荐

基于数据分析，生成3个选项：

```markdown
🎯 **今日学习推荐**

#### 选项1: 【继续项目】（推荐度: ⭐⭐⭐⭐⭐）
**内容**: 完成 [[项目名]] Phase X
**理由**: 
- 项目正在进行中，保持连续性
- Phase X涉及 [[概念A]]，与最近学习的 [[概念B]] 直接关联
- 查看 [[MOC-主题]] 进度: 已完成60%
**预计耗时**: X小时
**产出**: 练习记录 + 项目笔记更新

#### 选项2: 【复习巩固】（推荐度: ⭐⭐⭐⭐）
**内容**: 复习 [[概念C]]
**理由**:
- mastery=55，7天未复习（艾宾浩斯遗忘临界点）
- 该概念是 [[项目名]] 的前置知识
- 查看 [[MOC-主题]]: 该概念关联度较低，需要加强
**建议动作**: 完成 [[练习记录-概念C]]

#### 选项3: 【探索新知】（推荐度: ⭐⭐⭐）
**内容**: 学习 [[概念G]]
**理由**:
- 与已掌握的 [[概念H]] 高度关联（85%关联度）
- 是 [[项目名]] Phase Y 的前置知识
- 网络分析显示这是知识图谱的缺口
**预计耗时**: X小时

⚠️ **学习预警**
- 学习 [[概念G]] 时注意: 你曾犯错 [[MISTAKE-001]]（资源未释放）
- 建议提前复习错误档案
```

### 4. 记录学习决策

用户做出选择后，记录到 `.agent/profile.md`：

```markdown
## 当前学习计划

### 本次对话（YYYY-MM-DD）
- **用户选择**: 选项1 - 继续项目 [[项目名]] Phase X
- **AI推荐**: 选项1（匹配）
- **预计完成时间**: YYYY-MM-DD
- **关联知识点**: [[概念A]]、[[概念B]]

### 历史决策记录
- YYYY-MM-DD: 选择学习 [[概念A]]（实际完成✅）
- YYYY-MM-DD: 选择复习 [[概念B]]（实际完成❌，推迟）
```

## 推荐算法

### 推荐度计算因素

| 因素 | 权重 | 说明 |
|-----|------|------|
| 项目连续性 | 30% | 有进行中的项目优先 |
| 艾宾浩斯遗忘曲线 | 25% | 接近遗忘临界点的优先复习 |
| 知识关联度 | 20% | 与最近学习关联度高的优先 |
| 前置依赖 | 15% | 项目前置知识优先 |
| 网络密度 | 10% | 知识孤岛优先连接 |

### 预警规则

自动触发预警：
- mastery < 50 且 7天未复习 → "即将遗忘，建议复习"
- 历史错误模式匹配 → "注意避免 [[MISTAKE-XXX]]"
- 知识孤岛（入链+出链 < 2）→ "建议建立更多关联"
