## 🎯 核心理念

### 复利学习四原则
1. **每个知识点必须链接到已有知识** - 构建知识网络
2. **每个错误必须归类到错误模式库** - 避免重复踩坑
3. **每个项目必须产出可复用组件** - 积累实用资产
4. **每次对话必须沉淀为结构化笔记** - 固化学习成果

## 🚀 快速开始

1. **配置AI合伙人**：编辑 [AGENTS.md](AGENTS.md) 填写你的学习者画像
2. **查看学习路线**：参考 [LEARNING-ROADMAP.md](LEARNING-ROADMAP.md) 了解当前进度
3. **开始第一个项目**：进入 `01-Projects/project-01-console-bank/`

## 📚 使用指南

### 学习流程
```
学习新概念 → 实践项目 → 记录错误 → 更新知识库 → 生成练习 → 沉淀反思
```

### AI协作提示
- 每次对话前，AI会读取 [AGENTS.md](AGENTS.md) 了解你的学习背景
- 遇到错误时，AI会帮助归类到错误模式库
- 学习新内容时，AI会主动链接到已有知识
