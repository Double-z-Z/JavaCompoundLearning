# 原子笔记

> 知识网络的基本单元

---

## 定义

**原子笔记** = 只包含一个核心概念的笔记

特点：
- 单一概念，单一职责
- 可独立理解
- 通过链接与其他笔记连接
- 有 mastery 分数追踪掌握度

## 文件格式

```markdown
---
created: 2026-04-12
tags: [concurrency]
status: 🌿理解
mastery: 60
related_concepts:
  - [[CAS]]
  - [[分段锁思想]]
---

# 概念名称

## 一句话定义
用一句话概括这个概念

## 核心理解
详细解释...

## 关键关联
- [[相关概念A]] - 关联原因...
- [[相关概念B]] - 关联原因...

## 我的误区
- ❌ 误区...
- ✅ 纠正...
```

## 存放位置

```
02-Knowledge/<主题>/concepts/<概念名>.md
```

## 与深度文档的区别

| 类型 | 内容 | 长度 |
|-----|------|------|
| 原子笔记 | 单一概念 | 短（50-200行） |
| 深度文档 | 综合主题 | 长（500+行） |
