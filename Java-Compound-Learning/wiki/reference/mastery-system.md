# 掌握度系统

> 0-100分掌握度追踪

---

## 等级划分

| 分数 | 等级 | 图标 | 说明 |
|-----|------|------|------|
| 0-30 | 初识 | 🌱 | 刚了解，理解表面 |
| 31-60 | 理解 | 🌿 | 能解释概念 |
| 61-85 | 应用 | 🍎 | 能在项目中使用 |
| 86-100 | 掌握 | 🌳 | 能教授他人，处理边界情况 |

## 更新规则

| 活动 | 分数变化 | 说明 |
|-----|---------|------|
| 完成练习 | +10~20 | 验证理解 |
| 纠正错误 | +15~25 | 深度理解 |
| 建立新关联 | +10~15 | 扩展网络 |
| 教学/讲解 | +15~25 | 输出验证 |
| 发现盲区 | -5~10 | 理解偏差 |

## 存储位置

原子笔记的 frontmatter：
```yaml
---
mastery: 60
status: 🌿理解
---
```

profile.md 的汇总：
```yaml
mastered_concepts:
  - name: 线程池
    mastery: 75
    last_reviewed: 2026-04-10
```

## 遗忘曲线

基于艾宾浩斯遗忘曲线：
- mastery < 50 且 7天未复习 → 即将遗忘
- AI会在推荐中提示复习
