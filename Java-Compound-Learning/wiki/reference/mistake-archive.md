# 错误档案

> 将错误转化为知识资产

---

## 定义

错误档案 = 记录错误模式，避免重复犯错

价值：
- 同样的错误不犯第二次
- AI主动预警
- 错误成为学习机会

## 文件格式

```markdown
---
created: 2026-04-11
tags: [mistake, concurrency]
error_id: MISTAKE-001
status: active
---

# 错误标题

## 错误描述
### 现象
错误现象...

### 我的理解
当时的误解...

## 正确理解
正确的认知...

## 为什么错了
根本原因分析...

## 预防措施
1. 预防措施1
2. 预防措施2

## 关联知识
- [[概念A]]
- [[概念B]]
```

## 存放位置

```
03-Practice/mistakes/MISTAKE-<序号>-<描述>.md
```

## 工作流程

```
遇到错误
    ↓
错误分析（AI引导定位根因）
    ↓
错误归档（创建档案文件）
    ↓
强化练习（生成变式练习）
    ↓
下次学习时AI预警
```
