# 复利学习

> 核心理念详解

---

## 什么是复利学习？

复利学习借鉴金融学的复利概念：

- **本金** = 已有知识
- **利息** = 新知识通过与旧知识连接产生的增值
- **复利效应** = 知识网络越庞大，学习新知识越容易

## 传统学习 vs 复利学习

| 维度 | 传统学习 | 复利学习 |
|-----|---------|---------|
| 知识结构 | 线性（A→B→C） | 网络（A↔B↔C） |
| 遗忘速度 | 快 | 慢（互相强化） |
| 学习效率 | 恒定 | 加速（越学越快） |
| 知识应用 | 单点 | 组合创新 |

## 实现方式

1. **强制连接** - 每个新概念链接到至少2个已有概念
2. **定期复习** - 基于艾宾浩斯曲线在临界点复习
3. **项目整合** - 通过实战项目连接多个知识点
4. **错误归档** - 将错误转化为知识资产

## 量化指标

### 知识网络密度
```
网络密度 = 实际连接数 / 最大可能连接数

目标：10% → 50% → 80%
```

### 学习效率曲线
传统学习：线性增长  
复利学习：指数增长（加速）
