# Skill 系统

> AI的操作手册

---

## 定义

Skill = 定义AI在特定场景下行为的指令集

作用：
- 规范AI的响应方式
- 确保工作流一致性
- 复用最佳实践

## 文件结构

```
.trae/skills/<skill-name>/
└── SKILL.md
```

## 文件格式

```markdown
---
name: Skill名称
description: 触发时机和条件
---

# Skill标题

## 工作流程
1. 步骤1
2. 步骤2

## 输入/输出
### 输入
- 输入项1

### 输出
- 输出项1

## 示例
示例内容...
```

## 核心Skill列表

| Skill | 触发时机 | 职责 |
|-------|---------|------|
| 学习推荐 | "开始今日学习" | 分析状态，生成推荐 |
| 项目拆解 | 开始新项目 | 创建项目笔记和路径 |
| 整理笔记 | 对话结束 | 沉淀知识到知识库 |
| 错误分析 | 遇到错误 | 引导定位根因 |
| 错误归档 | 错误确认后 | 归档错误模式 |
| 强化练习 | 错误归档后 | 生成变式练习 |
| 苏格拉底式引导 | 深入理解 | 层层提问 |
| 横向拓展 | 理解困难 | 类比解释 |
